# hlzjzs
一款专为网文作者打造的极简、高效的沉浸式码字软件
极致沉浸式创作环境
无干扰模式 (Zen Mode)：一键 F11 进入全屏沉浸，隐藏所有 UI 元素，让您的视线只聚焦于文本。
打字机排版：正文底部预留呼吸感白区，让您的输入始终处于屏幕中央舒适视域。
平滑侧边栏：左侧章节目录支持一键箭头折叠，物理阻断外界视觉干扰。
2. 数据安全标准 (Stability First)
原子级防断电写入：无论系统断电还是突然死机，您的文稿永远不会出现“0 字节损坏”。
智能时光机 (History)：
动态捕获 100 份历史快照。
还原前置快照：在执行版本还原前，系统会强制生成一份“当前瞬时备份”，给后悔药再加上一层保险。
智能指纹缓存：针对百万字大长篇优化，基于文件修改时间的秒级索引，启动和切章如闪电般丝滑。
3. 高度可定制的“写稿手感”
全域色彩渲染：支持自定义护眼背景（米黄、浅绿）与各类个性文字色，光标与选区色彩智能适配。
顶尖中文字体支持：完美集成思源系列、马善政体等多种高颜值中文字库。
多维度版式调整：行高、字间距、加粗、字号均可精细调节，精准适配每个人的阅读习惯。
4. 生产力辅助工具
自定义每日码字目标。
全书/本章查找替换：支持全书范围内的原子级安全替换。
一键配置迁移：支持将所有设置一键导出到桌面，两秒钟在新电脑上找回手感。
健康提醒：每连续写作 1 小时自动弹出健康拉伸建议，守护您的视力与颈椎。
