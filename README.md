# HL Writer Assistant (HL写作助手)

HL Writer Assistant 是一款跨平台的写作辅助应用，专为创作者打造，提供专注、高效的写作环境。本项目基于 Flutter 构建。

## ✨ 核心特性

- **专注编辑器**: 支持首行缩进等符合中文写作习惯的排版体验。
- **章节管理**: 直观的侧边栏，轻松管理作品的大纲与章节结构。
- **历史回溯**: 自动保存写作历史记录，随时查看和恢复之前的版本，灵感不丢失。
- **便捷导出**: 支持将作品一键导出为常见格式。
- **全局搜索**: 强大的搜索功能，快速定位文章中的关键词。

## 🚀 快速开始

如果你想在本地运行此项目，请确保你已安装了 [Flutter SDK](https://flutter.dev/docs/get-started/install)。

### 1. 克隆项目
```bash
git clone [https://github.com/BZLKK/hlzjzs.git](https://github.com/BZLKK/hlzjzs.git)
```
cd hlzjzs
2. 获取依赖
```bash
flutter pub get
```
3. 运行应用
```bash
flutter run
```
